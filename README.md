# hsnews

A [Hacker News](http://news.ycombinator.com) clone written in [Clojure](http://clojure.org) using [Noir](http://webnoir.org).

Created for [Hacker School](http://hackerschool.com) batch[2].

## Usage

    lein deps
    lein run
